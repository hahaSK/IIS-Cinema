Instalace

Stručně popiąte:
  • postup instalace na server,
  • Potřebné balíčky: npm verze 6.9.0, node verze 10.16.0 a python3.6. Pro ostatné balíčky viď requirements.txt
  • Pro poľadavky vid. requirements.txt. Případne stačí zadat pip install -r requirements.txt
  • Pouľití:
    • Jako první je potřebné si stáhnout/naklonovat repozitář.
    • Je vhodné si nainstalovat a vytvořit virtual environment pomocí příkazu sudo python -m pip install virtualenv a virtualenv -p python3.6 venv
    • Pak se přepnout do virtual environment-u pomocí . venv/bin/activate pouľit výąe uvededej příkaz pro instalaci potřebných balíčků.
    • V settings.py je moľní zvolit lokálnou databázy pracující na postgre enginu.
    • Migrace sa vytvářejé pomocí manage.py makemigrations a pak manage.py migrate
    • Frontendové závislosti se nainstalují pomocí npm install a pak je moľní spustit aplikaci pomocí npm start
